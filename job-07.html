PENGGUNAAN TAG-TAG PADA HTML

Untuk memperoleh suatu tampilan yang menarik dalam dokumen HTML,diperlu-

kan kemampuan khusus salah satunya yaitu kemampuan untuk memformat/meng-

atur teks pada dokumen HTML.


Untuk melakukan pengaturan teks tidak terlepas dari penggunaan tag-tag HTML seperti:

1. Tag tulisam : untuk menampilkan tulisan miring

2. Tag tulisan : untuk menampilkan tulisan tulisan

3. Tag tulisan : untuk menampilkan tulisan tebal

4. tag tulisan : untuk menampilkan tulisan lebih kecil dari ukuran normal

5. Tag tulisan : untuk menampilkan tulisanlebih besar dari ukuran normal

6. Tag tulisam : untuk menampilkan tulisan seperti tulisammesin ketik

7. Tag tulisan : untuk menampilkan tulisanbergaris bawah

8. tag tulisan : untuk menampilkan tulisan menjadisubcript

9. Tag tulisan : untuk menampilkan tulisan menjadisuperscript
